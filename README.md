Project Overview

The Campus Event Reporting Dashboard is a web-based analytics system designed to monitor and analyze student events across multiple colleges.
It provides interactive visualizations and performance metrics to help administrators understand event popularity, student participation, attendance rates, and feedback scores.

The dashboard includes dynamic filters, KPI summaries, charts, and ranking tables for quick and efficient reporting.
